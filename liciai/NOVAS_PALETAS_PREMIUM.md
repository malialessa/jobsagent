## 🌟 Opção 1: **Executive Dark** (Recomendada)
*Inspirada em: Bloomberg Terminal, Goldman Sachs, Notion Dark*

### Cores Principais
```css
/* Backgrounds */
--bg: #0D0D0F          /* Preto profundo */
--bg2: #16171A         /* Grafite escuro */
--panel: #1C1E22       /* Painel grafite */
--panel2: #111214      /* Painel profundo */

/* Acentos */
--primary: #00D9FF     /* Cyan elétrico */
--secondary: #0EA5E9   /* Azul profissional */
--accent: #10B981      /* Verde sucesso */
--warning: #F59E0B     /* Âmbar avisos */

/* Gradientes */
--gradient-primary: linear-gradient(135deg, #00D9FF 0%, #0EA5E9 100%)
--gradient-success: linear-gradient(135deg, #10B981 0%, #34D399 100%)
--gradient-hero: linear-gradient(135deg, #00D9FF 0%, #10B981 100%)

/* Sombras */
--shadow-primary: 0 8px 24px rgba(0, 217, 255, 0.25)
--shadow-success: 0 8px 20px rgba(16, 185, 129, 0.20)
```

### Uso no Sistema
- **ScoreBar 80-100**: Gradiente verde sucesso brilhante
- **ScoreBar 60-79**: Gradiente cyan → azul profissional
- **ScoreBar <60**: Gradiente cinza slate neutro
- **Pills selecionados**: Cyan elétrico com glow azul
- **Hover rows**: Border-left cyan + overlay verde translúcido
- **Chips**: Background cyan 12%, texto cyan brilhante
- **Badges**: Cyan para info, verde para sucesso

### Preview
```

│  🎯 Score 85    ✅ [SP] [RJ]  ⚡ 2 filtros │  ← Cyan elétrico
│  ━━━━━━━━━━━━━░░░░░░░░░░░                 │  ← Barra verde sucesso
│                                            │
│  ┃  Pregão Eletrônico - R$ 2.5M          │  ← Hover = cyan
│  ┃  Encerra: 27/03  |  Dias: 3d ⚠️       │  ← Âmbar nos urgentes
└────────────────────────────────────────────┘
```

**Vantagens**:
- ✅ Cyan elétrico = Tecnologia + Modernidade (Bloomberg usa)
- ✅ Verde = Oportunidades + Sucesso (psicologia positiva)
- ✅ Azul profissional = Confiança corporativa
- ✅ Alto contraste WCAG AAA com fundo grafite
- ✅ Visual sério e executivo
